JavaScript Basics

Javascript is a cross-platform, object-oriented scripting language. Javascript is extremely popular for a variety of reasons. It is a small and lightweight language allow maximum flexibility for developers to take it in a bunch of different directions. JavaScript lives inside a host eviroment (a web browser or Node Server), it can be conntected to the objects of these enviroments to provide programmatic control over them.

  • Variable Declaration Javascript variables are containers for storing data values - imagine a cup you fill with coffee, the cup holds the coffee, a variable holds a value. All JavaScript varibles must be identified with unique names. These unique names are called identifiers.
    var x;

  • Variable Assignment Assignment operators assign vaues to JavaScript varibales - our cup now can have coffee poured in it, giving our variable a value to hold. The = assignment operator assigns a value to a variable.
    var=10;
  • Data Type Data types are important concept; to be able to operate on variable yu need to know the data type. There are six data type. There are six data types that are JavaScript primitives: Boolean - true or false; null aka nothing; Number -
    42

    String -

    Array -
    [1, 'Coding', 2, 'Dojo'];
    and Object -
